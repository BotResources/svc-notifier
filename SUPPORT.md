# Support

**There is no public support channel for this repository.**

- This code is published read-only. We do not provide support, troubleshooting,
  Q&A, or community help through GitHub.
- Issues and Discussions on this repository are disabled. Requests posted by any
  other means will be closed without response.
- Clients with a commercial relationship with BotResources: contact your BR
  account manager.

For security reports, see [SECURITY.md](SECURITY.md).
